## Запуск

```bash
./run.sh build_generator
./run.sh run_generator
./run.sh build_reporter
./run.sh run_reporter
```

После этого в `data/` появятся `data.csv` и `report.html`.

## Как открыть отчёт в браузере (GitHub Codespaces)
Цепочка проброса до браузера на компьютере состоит из трёх уровней:

1. nginx слушает порт 80 внутри контейнера;
2. `docker run -p 8080:80` публикует этот порт на хосте Codespaces как порт 8080;
3. Codespaces обнаруживает слушающий порт 8080 и пробрасывает его через свой
   прокси по адресу вида `https://<codespace>-8080.app.github.dev`, доступному
   из браузера на компьютере.

Порядок действий:

1. Открыть репозиторий в GitHub Codespaces.
2. Выполнить запуск (см. выше), чтобы получить `data/report.html`.
3. Запустить веб-сервер:

   ```bash
   ./run.sh report_server
   ```
4. В VS Code открыть вкладку **PORTS**. Порт 8080 пробрасывается автоматически.
   Если его нет — нажать **Forward a Port** и указать `8080`.
5. В строке порта 8080 нажать на значок глобуса (**Open in Browser**) или
   скопировать **Forwarded Address**. Отчёт откроется в браузере.
